# Neighborhood Map
This is the last project in Udacity Nanodegree Front End Web Developer by using React and the Google Maps API.


# Folder Structure

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    App.css
    App.js
    App.test.js
    index.css
    index.js
    locations.js
    mapCustomStyle.js
```

# How to Start

In command line enter project folder name
cd neighborhoodmap
and type
npm Start
now you can view project in your browser with address
http://localhost:3000/
